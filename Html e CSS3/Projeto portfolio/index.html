Sobre o Projeto

Este site foi desenvolvido como um desafio prático do curso de HTML5 e CSS3 do canal Curso em Vídeo, sob orientação do professor Gustavo Guanabara.

O que foi aplicado:
  • Efeito Parallax: Uso de imagens de fundo fixas (background-attachment: fixed) que criam profundidade durante a rolagem.
  • Design Responsivo: O layout se adapta perfeitamente a diferentes tamanhos de tela, desde celulares até desktops.
  • Tipografia: Aplicação de fontes personalizadas via Google Fonts para remeter à estética da literatura de cordel.
  • Semântica HTML5: Organização do conteúdo utilizando as tags corretas para melhor acessibilidade e SEO.

O texto utilizado no projeto é o poema "Cordel Moderno" de Milton Duarte, que faz uma reflexão sobre a tecnologia e a tradição.

Você pode conferir o código fonte e outros projetos no meu GitHub.

Projeto Android

Saiba mais sobre o Projeto Android
Sobre o Projeto

Este site foi desenvolvido como um desafio prático do curso de HTML5 e CSS3 do canal Curso em Vídeo, sob orientação do professor Gustavo Guanabara.

O que foi aplicado:
  • Design Responsivo: Uso de técnicas de Media Queries para garantir que o site seja perfeitamente legível em telas de celulares, tablets e computadores.
  • Componentes Flexíveis: Criação de layouts adaptáveis e menus que se ajustam conforme a largura do dispositivo.
  • Variáveis CSS: Organização do código utilizando :root para padronizar a paleta de cores e fontes do tema Android.
  • Semântica e Mídia: Uso correto de tags semânticas e incorporação de vídeos e imagens de forma otimizada para a web.

O projeto reconta a história do mascote do Android e como o sistema operacional do Google surgiu, apresentando as curiosidades do "Bugdroid".

Você pode conferir o código fonte e outros projetos no meu GitHub.
